🤖 AI Chatbot Widget

Okul Chatbot'u Nasıl Çalışır?

Sol alt köşedeki 💬 butonuna tıklayarak chatbot'u açabilirsiniz. Chatbot, okul web sitesinden (turktelekomatl.meb.k12.tr) güncel bilgileri çeker ve Groq yapay zeka API'si ile soruları yanıtlar.

Veriler 30 dakikada bir otomatik güncellenir. Manuel güncelleme için /api/scrape endpoint'ini çağırabilirsiniz.

📦 Okul Sitesine Ekleme Kodu

Aşağıdaki kodu okul web sitesinin </body> etiketinin hemen üstüne ekleyin:

<!-- TT MTAL AI Chatbot Widget --> <script src="http://SUNUCU_ADRESI:3000/chatbot-widget.js"></script>

⚠️ SUNUCU_ADRESI kısmını API sunucunuzun gerçek IP/domain adresiyle değiştirin.

🔌 API Endpoint'leri

Chatbot API sunucusu aşağıdaki endpoint'leri sunar:

POST /api/chat Sohbet mesajı gönder
GET /api/scrape Verileri güncelle
GET /api/status Sunucu durumu
GET /api/data Ham okul verisi
